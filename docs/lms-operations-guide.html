LMS + Slack Supervisor Integration Guide

Runtime configuration

Use deployment-time environment values instead of hardcoded production endpoints and credentials. 

{
  "webhookUrl": "{{N8N_WEBHOOK_URL}}",
  "alertEmail": "{{LMS_ALERT_EMAIL}}",
  "backupFolderId": "{{BACKUP_FOLDER_ID}}",
  "slackBotToken": "{{SLACK_BOT_TOKEN}}"
}

Example shell configuration

export N8N_WEBHOOK_URL="{{N8N_WEBHOOK_URL}}"
export LMS_ALERT_EMAIL="{{LMS_ALERT_EMAIL}}"
export BACKUP_FOLDER_ID="{{BACKUP_FOLDER_ID}}"
export SLACK_BOT_TOKEN="{{SLACK_BOT_TOKEN}}"
